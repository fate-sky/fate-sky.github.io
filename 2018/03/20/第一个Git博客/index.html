第一个Git博客

你知道什么是啪啪啪的意思吗？和嘿嘿嘿一样。笑话内容：有个男人去减肥中心，交了500块钱，而减肥的方法是——穿着三点式的美女，和男人在屋子里跑圈追逐。那个美女是这样讲的：“你追我，如果你追到我，我就让你嘿嘿嘿。”
旁边的男主持不依不饶：嘿嘿嘿是什么意思？
费玉清突然很腼腆：就是那个那个嘛！！……
故事发展到后来，男人追到了美女，嘿嘿嘿之后效果很好，于是他就继续花钱嘿嘿嘿，结果最后没有忍住尝试了最贵的那一款……
然后发现居然变成了一只大猩猩在屋里，坐着对他说：“我追你，如果我追到你，你就让我嘿嘿嘿。”
